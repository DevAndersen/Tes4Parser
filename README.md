# Tes4Parser

A .NET parser for the TES4 file format.

Special thanks to the editors of [The Unofficial Elder Scrolls Pages (UESP)](https://uesp.net), specifically for [their documentation of the TES4 file format](https://en.uesp.net/wiki/Skyrim_Mod:Mod_File_Format), without which this project would not be possible.
